1. "Der Entscheidungs-Detektiv"
   - Ziel: Wähle in 5 Fällen jeweils die mathematisch beste Option aus.
   - Spielmechanik: 
     - Szenario: "Wähle den besten Fluchtwagen." (Attribute: Speed, Panzerung, Preis).
     - Der Spieler setzt Gewichtungen (z.B. Speed 50%, Preis 30%).
     - Er muss dann aus 3 Autos dasjenige wählen, das nach SEINER Gewichtung rechnerisch gewinnt.
     - Messbarkeit: Wählt er das rechnerisch falsche Auto (z.B. Bauchgefühl sagt Porsche, Rechnung sagt Volvo) -> 0 Punkte. Wählt er das mathematisch korrekte -> 100 Punkte. 
   - Lernziel: Verständnis, dass NWA eine objektive Entscheidungshilfe ist und man dem Rechenweg folgen muss.

2. "WG-Zimmer-Challenge"
   - Ziel: Finde unter 10 Angeboten die eine Wohnung, die mehr als 450 Nutzwertpunkte erreicht.
   - Spielmechanik: 
     - Liste von Wohnungen mit Werten (Miete 500€, 20qm...).
     - Vorgegebene Gewichtung (Preis 60%, Größe 40%).
     - Der Spieler muss im Kopf oder mit Taschenrechner überschlagen und auf die *EINE* richtige Wohnung klicken.
     - Messbarkeit: Richtiger Klick = 100%. Falscher Klick = 0% (oder 2. Versuch = 50%).
   - Lernziel: Anwendung der Berechnung "Wert * Gewicht".

3. "Der Matrix-Architekt"
   - Ziel: Baue die NWA-Tabelle fehlerfrei zusammen.
   - Spielmechanik: 
     - Drag & Drop Puzzle.
     - Bausteine: "Kriterium", "Gewichtung", "Note", "Teilnutzwert", "Summe".
     - Der Spieler muss die Spaltenköpfe und die Rechenoperationen (z.B. "Gewichtung * Note = ?") an die richtige Stelle ziehen.
     - Messbarkeit: Jedes korrekt platzierte Element gibt Punkte. Am Ende %-Score der korrekten Struktur.
   - Lernziel: Verinnerlichung des Tabellenaufbaus.

4. "Fehler-Suche: Der korrupte Analyst"
   - Ziel: Finde alle 5 Rechenfehler in einer fertigen NWA innerhalb von 90 Sekunden.
   - Spielmechanik: 
     - Eine komplette Tabelle wird gezeigt.
     - Manche Werte stimmen nicht (z.B. Gewichtung 30% * Note 10 = 300 statt 3).
     - Spieler klickt auf Zellen.
     - Messbarkeit: Gefundener Fehler = +20%. Falscher Klick (korrekte Zelle) = -10%. Ziel: 100%.
   - Lernziel: Kritische Prüfung von Ergebnissen, Genauigkeit.

5. "Kunden-Wunsch-Rechner"
   - Ziel: Erreiche mindestens 90% Kundenzufriedenheit durch korrekte Gewichtung.
   - Spielmechanik: 
     - Kunde sagt: "Preis ist mir extrem wichtig, Marke egal."
     - Spieler muss Schieberegler einstellen: Preis auf 80-90%, Marke auf 0-10%.
     - Das Spiel berechnet die Abweichung vom "Idealprofil" des Kunden.
     - Messbarkeit: 
       - Abweichung < 10% -> 100 Punkte.
       - Abweichung < 20% -> 80 Punkte.
       - Sonst -> 0 Punkte ("Kunde kauft woanders").
   - Lernziel: Transfer von verbalen Anforderungen in numerische Prozentwerte.
