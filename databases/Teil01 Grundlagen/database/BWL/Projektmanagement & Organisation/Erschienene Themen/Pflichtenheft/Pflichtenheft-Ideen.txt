1. "Pflichtenheft-Check"
   - Ziel: Unterscheide 10 Inhalte: Gehören sie ins Pflichtenheft oder Lastenheft?
   - Spielmechanik: 
     - Inhalt: "Datenbank-Design (ERM)". -> Pflichtenheft (WIE).
     - Inhalt: "Ziel: Umsatzsteigerung". -> Lastenheft (WAS/WOZU).
     - Inhalt: "GUI-Mockups". -> Pflichtenheft (Lösung).
     - Messbarkeit: Korrekte Zuordnung.
   - Lernziel: Abgrenzung der Dokumente.

2. "Lösungs-Architekt"
   - Ziel: Formuliere eine technische Lösung für eine Anforderung.
   - Spielmechanik: 
     - Anforderung (Lastenheft): "System muss mobil verfügbar sein."
     - Lösungswahl (Pflichtenheft): "Wir entwickeln eine native iOS-App" vs "Wir kaufen PCs".
     - Messbarkeit: Passende technische Antwort auf die Anforderung.
   - Lernziel: Übersetzung von Anforderungen in Lösungen.

3. "Abnahme-Kriterien-Definierer"
   - Ziel: Lege fest, wann die Arbeit im Pflichtenheft als "erledigt" gilt.
   - Spielmechanik: 
     - Feature: "Login".
     - Kriterium A: "Login geht irgendwie." (Zu vage).
     - Kriterium B: "User kann sich mit Email/PW einloggen und sieht Dashboard." (Messbar).
     - Spieler wählt B.
     - Messbarkeit: Qualität der Kriterien.
   - Lernziel: Definition von "Done".

4. "Genehmigungs-Lauf"
   - Ziel: Hole die Unterschrift vom Auftraggeber.
   - Spielmechanik: 
     - Dokument wird "eingereicht".
     - Kunde hat Fragen (Rücksprache). Spieler muss antworten.
     - Kunde: "Ist das im Preis drin?" -> Spieler: "Ja/Nein" (Verweis auf Vertrag).
     - Ziel: Unterschrift sicherstellen.
     - Messbarkeit: Anzahl der Iterationen bis zur Unterschrift.
   - Lernziel: Pflichtenheft als Vertragsgrundlage.

5. "Änderungs-Manager (Change Request)"
   - Ziel: Entscheide korrekt bei nachträglicher Änderung.
   - Spielmechanik: 
     - Pflichtenheft ist unterschrieben.
     - Kunde will "Doch noch eine Suchfunktion".
     - Option A: "Ok, mach ich schnell." (Falsch! Scope Creep).
     - Option B: "Change Request schreiben + Mehrkosten anbieten." (Richtig).
     - Messbarkeit: Korrektes Prozessverhalten.
   - Lernziel: Umgang mit Änderungen nach Vertragsabschluss.
