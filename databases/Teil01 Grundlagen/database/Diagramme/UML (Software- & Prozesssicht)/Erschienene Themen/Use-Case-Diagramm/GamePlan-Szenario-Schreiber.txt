5. "Szenario-Schreiber"
   - Ziel: Vom Bild zum Text.
   - Spielmechanik: 
     - Use Case "Geld abheben".
     - Schreibe den "Main Success Scenario" (Happy Path).
     - 1. Karte rein. 2. PIN ok. 3. Betrag wählen. 4. Geld raus.
     - Schreibe eine "Extension" (Alternative).
     - 2a. PIN falsch -> Abbruch.
     - Messbarkeit: Logischer Textablauf passend zum Oval.
   - Lernziel: Detaillierung von Anwendungsfällen.
