1. "Hardware-Profiler"
   - Ziel: Stelle den optimalen PC für das Szenario zusammen.
   - Spielmechanik: 
     - Szenario: "CAD-Workstation". -> Braucht: Starke GPU (Quadro), Viel RAM, Teuer.
     - Szenario: "Office-PC". -> Braucht: i3/i5, SSD, Leise, Günstig.
     - Szenario: "Gaming-PC". -> Braucht: RGB, Starke GPU (GeForce), Gute Kühlung.
     - Messbarkeit: Preis-Leistungs-Verhältnis und Erfüllung der Anforderungen.
   - Lernziel: Anforderungsprofile erkennen.

2. "Mobile vs. Stationär"
   - Ziel: Entscheide: Laptop, Tablet oder Desktop?
   - Spielmechanik: 
     - Anforderung: "Außendienst, Unterschriften beim Kunden". -> Tablet.
     - Anforderung: "Videobearbeitung im Büro, 4 Monitore". -> Desktop.
     - Anforderung: "Student, Bibliothek & Hausarbeit". -> Laptop.
     - Messbarkeit: 5 Szenarien korrekt zugeordnet.
   - Lernziel: Formfaktoren und Einsatzzwecke.

3. "Server-Typen"
   - Ziel: Was ist das? (Tower, Rack, Blade).
   - Spielmechanik: 
     - Bild zeigt flachen Server zum Einschieben. -> Rack-Server.
     - Bild zeigt Server, der aussieht wie normaler PC. -> Tower-Server.
     - Bild zeigt Modul für Chassis. -> Blade.
     - Messbarkeit: Korrekte Benennung.
   - Lernziel: Bauformen im Rechenzentrum.

4. "Micro-Computer-Bastler"
   - Ziel: Was kann ein Raspberry Pi?
   - Spielmechanik: 
     - Aufgabe: "Digital Signage Anzeige steuern (günstig)".
     - Wahl: PC (500€) vs Pi (50€). -> Pi gewinnt.
     - Aufgabe: "High-End Gaming". -> PC gewinnt.
     - Messbarkeit: Wirtschaftlichste Hardware gewählt.
   - Lernziel: Einsatzgebiete von Einplatinencomputern.

5. "Das Peripherie-Quiz"
   - Ziel: Ein- oder Ausgabegerät? (EVA-Prinzip).
   - Spielmechanik: 
     - Objekt: Maus. -> Eingabe.
     - Objekt: Monitor. -> Ausgabe.
     - Objekt: Touchscreen. -> Ein- und Ausgabe (Kombiniert).
     - Messbarkeit: 100% korrekte Sortierung.
   - Lernziel: Grundlagen EVA.
