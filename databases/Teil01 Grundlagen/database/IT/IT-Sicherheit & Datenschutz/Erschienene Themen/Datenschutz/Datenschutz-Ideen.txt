1. "Personenbezogen oder nicht?"
   - Ziel: Sortiere Daten in "Schützenswert (DSGVO)" und "Unkritisch".
   - Spielmechanik: 
     - "IP-Adresse". -> Personenbezogen.
     - "Maschinen-Seriennummer". -> Unkritisch.
     - "Email-Adresse info@firma.de". -> Unkritisch (Juristische Person).
     - "Email-Adresse hans.meier@firma.de". -> Personenbezogen.
     - Messbarkeit: 100% korrekte Trennung.
   - Lernziel: Definition personenbezogener Daten (Art. 4 DSGVO).

2. "Rechte-Memory"
   - Ziel: Finde die Rechte der Betroffenen.
   - Spielmechanik: 
     - Icon "Auge" -> Recht auf Auskunft (Art. 15).
     - Icon "Mülleimer" -> Recht auf Löschung (Art. 17).
     - Icon "Stift" -> Recht auf Berichtigung (Art. 16).
     - Messbarkeit: Paare aus Icon und Artikel finden.
   - Lernziel: Betroffenenrechte kennen.

3. "Bußgeld-Richter"
   - Ziel: Verhängen Sie das Bußgeld im richtigen Rahmen (2% oder 4% / 10 Mio oder 20 Mio).
   - Spielmechanik: 
     - Verstoß: "Passwort-Zettel am Monitor". -> Kleiner Verstoß.
     - Verstoß: "Datenleck von 1 Mio Kunden vertuscht". -> Max Strafe (4% Jahresumsatz).
     - Messbarkeit: Verhältnismäßigkeit der Strafe.
   - Lernziel: Drohpotential der DSGVO.

4. "Verfahrensverzeichnis-Builder"
   - Ziel: Dokumentiere eine Verarbeitungstätigkeit korrekt.
   - Spielmechanik: 
     - Prozess: "Gehaltsabrechnung".
     - Felder: "Zweck", "Kategorie Daten", "Löschfrist", "Empfänger".
     - Spieler zieht "Finanzamt" auf "Empfänger". "10 Jahre" auf "Löschfrist".
     - Messbarkeit: Vollständiges VVT.
   - Lernziel: Dokumentationspflichten (Art. 30).

5. "Der Auftragsverarbeiter"
   - Ziel: Prüfe, ob ein AV-Vertrag nötig ist.
   - Spielmechanik: 
     - "Wir hosten Webseite bei Provider X". -> AV-Vertrag nötig.
     - "Der Postbote bringt Briefe". -> Kein AV-Vertrag (Postgeheimnis).
     - "Putzfirma leert Papierkörbe". -> Verschwiegenheitserklärung (aber kein AV).
     - Messbarkeit: Korrekte juristische Einordnung.
   - Lernziel: Unterscheidung Auftragsverarbeitung vs. Fremdleistung.
