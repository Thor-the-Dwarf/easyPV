Anforderungen
3. "Responsive-Tester"
   - Ziel: Mach es mobilfreundlich.
   - Spielmechanik: 
     - Webseite auf Desktop: 3 Spalten nebeneinander.
     - Webseite auf Handy: Immer noch 3 Spalten (viel zu klein!).
     - Lösung: Media Query `@media (max-width: 600px)`.
     - Änderung: `flex-direction: column`.
     - Ergebnis: Spalten untereinander.
     - Messbarkeit: Lesbarkeit auf kleinen Screens.
   - Lernziel: Media Queries.
