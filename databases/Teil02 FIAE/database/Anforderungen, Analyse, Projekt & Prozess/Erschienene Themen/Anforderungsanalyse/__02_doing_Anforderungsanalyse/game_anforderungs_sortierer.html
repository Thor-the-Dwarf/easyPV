Anforderungs-Sortierer

AP2-Training: Ordne Anforderungen sicher ein.

Aufgabe: Ordne jede Aussage als funktionale (FKT) oder nicht-funktionale Anforderung (NFA) ein.

FKT (funktional): Was das System koennen muss.

NFA (nicht-funktional): Wie gut, sicher, schnell oder stabil es laufen muss.

Beispiel: "User kann Passwort zuruecksetzen" = FKT, "Antwortzeit unter 500 ms" = NFA.
