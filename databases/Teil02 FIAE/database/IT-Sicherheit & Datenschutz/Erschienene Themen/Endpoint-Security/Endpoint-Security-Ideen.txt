1. "Malware-Scanner-Match"
   - Ziel: Schutz von Endgeräten vor schädlicher Software.
   - Spielmechanik: 
     - Ordne Bedrohungen (Virus, Trojaner, Ransomware) den passenden Schutzmaßnahmen zu.
     - Lernziel: Bedrohungslandschaft am Endpoint.

2. "Firewall-Rule-Builder"
   - Ziel: Kontrolliere eingehenden und ausgehenden Datenverkehr.
   - Spielmechanik: 
     - "Erlaube Port 443, blockiere Port 21". 
     - Schütze den PC vor unbefugten Zugriffen.
     - Lernziel: Netzwerk-Sicherheit am Client.

3. "USB-Lock-Down"
   - Ziel: Verhindere Datenabfluss über Hardware.
   - Spielmechanik: 
     - Ein unbekannter USB-Stick wird eingesteckt. Was tust du? (Blockieren).
     - Lernziel: Device Control Richtlinien.

4. "Patch-Management-Racer"
   - Ziel: Halte Systeme aktuell.
   - Spielmechanik: 
     - "Kritisches Update verfügbar". Installiere es, bevor der Exploit zuschlägt.
     - Lernziel: Schwachstellenmanagement.

5. "Shadow-IT Detective"
   - Ziel: Identifiziere nicht autorisierte Software.
   - Spielmechanik: 
     - Finde Programme auf dem Rechner, die nicht vom Admin freigegeben wurden.
     - Lernziel: Endpoint Compliance.
