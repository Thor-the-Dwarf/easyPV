1. "Environment-Variable-Quest"
   - Ziel: Trenne Konfiguration von Code.
   - Spielmechanik: 
     - Verschiebe einen API-Key aus der `config.js` in eine `.env` Datei.
     - Lernziel: Best Practices für Secrets.

2. "Git-Ignore-Guardian"
   - Ziel: Verhindere versehentliche Uploads von Passwörtern.
   - Spielmechanik: 
     - Füge `.env` und `credentials.json` zur `.gitignore` hinzu.
     - Lernziel: Schutz des Repositories.

3. "Secret-Rotation-Timer"
   - Ziel: Minimiere den Schaden bei Leaks.
   - Spielmechanik: 
     - Ein Key wurde geleakt. Erzeuge so schnell wie möglich einen neuen und tausche ihn überall aus.
     - Lernziel: Incident Response für Credentials.

4. "Vault-vs-Hardcoded Match"
   - Ziel: Nutze professionelle Tools.
   - Spielmechanik: 
     - Vergleiche: Key im Code (Unsicher) vs. Key im Azure Key Vault / HashiCorp Vault.
     - Lernziel: Enterprise Security.

5. "Leak-Detector-Scanner"
   - Ziel: Finde vergessene Keys in alten Commits.
   - Spielmechanik: 
     - Nutze Tools wie "TruffleHog" (simuliert), um die History zu scannen.
     - Lernziel: Altlasten-Bereinigung.
