Anforderungen (UX-Game-Optimiert)
- Ziel: Sicherer Umgang mit API-Schnittstellenbeschreibungen (Schemas) und Validierung von Request/Response Objekten.
- Game-Mechanic: "The API Border Control". Du bist der Grenzbeamte an einem Hyper-Speed Datenhafen. Datenpakete (JSON) rasen heran. Du hast das "Gesetzbuch" (Das Schema). Du musst jedes Paket scannen. Passt ein Typ nicht (`String` statt `Int`) oder fehlt ein `Required`-Feld? ABGEWIESEN!
- Challenge: 
  - Level 1 (Basics): Einfache Typ-Checks. "Alter als Zahl erwartet".
  - Level 2 (Structure-Check): Fehlende Pflichtfelder in verschachtelten Objekten (`address.zip` fehlt).
  - Level 3 (Regex-Rallye): Validiere Formate wie E-Mails oder Datums-Strings gegen den Vertrag unter extremem Zeitdruck.

Arbeitspakete (UX-Game-Optimiert)
1. Interactive HUD und Scoring
- [ ] HUD-Elemente: "Border-Security-Level", "Packet-Frequency", "Scan-Accuracy".
- [ ] Scoring: "Secure-Pass" Bonus für schnelles Akzeptieren korrekter Pakete. Abzug für "Protocol-Violation" (falsche Entscheidung).

2. Screen-Layout (The Terminal)
- [ ] `dock-bay -> schema-hologram -> payload-scanner -> reject-bin`.
- [ ] Schema-Hologram: Ein statischer Referenz-Code-Block auf der linken Seite.
- [ ] Payload-Scanner: Ein fließender Strom von JSON-Karten, die in der Mitte pausieren.

3. Kernvisuals und Effekte
- [ ] Cyber-Airport Ästhetik: Kaltes Neon-Blau, transparente UI-Panels, ASCII-Art Scanner-Linien, digitale Störgeräusche bei Fehlern.
- [ ] "Laser-Scan": Eine horizontale Laserlinie fährt über den JSON-Code. Fehlerhafte Zeilen glühen rot auf.
- [ ] "Data-Stream": Hintergrund-Animationen von binärem Code, der vorbeihuscht.

4. Browser- und Device-Resilienz
- [ ] `font-family: 'Courier New', monospace` für den authentischen API-Look.
- [ ] Mobile: "Swipe Left (Reject) / Swipe Right (Accept)" Steuerung.

5. Content-Strategie (The Payloads)
- [ ] 20 reale API-Bodies: Von "Google-Auth-Response" bis "IoT-Sensor-Ping".

6. QA-Paket
- [ ] Test: Sind "Missing-Field" Fehler auch bei tief verschachtelten JSONs auf Mobile gut sichtbar (Auto-Scroll-to-Error)?
- [ ] Test: Reagiert das Haptic-Feedback bei "Reject" passend zur Cyber-Ästhetik?

7. Abnahme-Kriterien
- [ ] Der Spieler verankert das Prinzip der strikten Typisierung und Pflichtfelder.
- [ ] Das Interface wirkt wie ein futuristisches Tool für IT-Security Experten.
- [ ] Hoher Lerneffekt für OpenAPI und JSON-Schema.
