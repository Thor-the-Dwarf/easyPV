1. "Breakpoint-Switcher"
   - Ziel: Verstehe Media Queries.
   - Spielmechanik: 
     - Verändere die Fensterbreite. 
     - Bei 768px: Das Menü muss zum Hamburger-Icon werden.
     - Lernziel: Layout-Anpassung an verschiedene Geräte.

2. "Mobile-First Puzzle"
   - Ziel: Priorisiere Inhalte für kleine Bildschirme.
   - Spielmechanik: 
     - Du hast 10 Elemente, aber nur Platz für 3 auf dem Smartphone.
     - Was versteckst du im "Mehr"-Menü?
     - Lernziel: Progressive Advancement & Graceful Degradation.

3. "Flexbox-Froggy-Clone"
   - Ziel: Übe die Ausrichtung mit CSS Flexbox.
   - Spielmechanik: 
     - Positioniere Container mit `justify-content` und `align-items`.
     - Lernziel: Modernes CSS Layouting.

4. "View-Port-Check"
   - Ziel: Nutze relative Einheiten (vh, vw, rem).
   - Spielmechanik: 
     - Textgröße in `px` vs. `rem`. 
     - Was passiert, wenn der User die Systemschriftart zoomt?
     - Lernziel: Skalierbarkeit von Layouts.

5. "Touch-Target-Master"
   - Ziel: Optimiere Abstände für Finger-Bedienung.
   - Spielmechanik: 
     - Klicke auf nebeneinanderliegende Links. 
     - Wenn sie zu nah sind -> Fehlklick.
     - Lernziel: 44x44 Pixel Regel für Touch-UIs.
