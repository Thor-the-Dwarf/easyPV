1. "XML-Envelope-Puzzle"
   - Ziel: Verstehe den Aufbau einer SOAP-Nachricht.
   - Spielmechanik: 
     - Envelope -> Header -> Body.
     - Lernziel: Struktur von SOAP.

2. "WDSL-Explorer"
   - Ziel: Nutze die Dienstbeschreibung.
   - Spielmechanik: 
     - Lies das WSDL-File und finde heraus, welche Methoden der Service anbietet.
     - Lernziel: Selbstbeschreibende Schnittstellen.

3. "SOAP-vs-REST-Match"
   - Ziel: Kenne die Unterschiede.
   - Spielmechanik: 
     - Welches nutzt XML? (SOAP). Welches ist zustandslos? (Beide, aber REST stärker).
     - Lernziel: Technologie-Vergleich.

4. "RPC-Simulator"
   - Ziel: Verstehe Remote Procedure Calls.
   - Spielmechanik: 
     - Rufe eine Funktion auf einem fremden Server auf, als wäre sie lokal.
     - Lernziel: Abstraktionsschicht von SOAP.

5. "Error-Handling-SOAP"
   - Ziel: Verstehe SOAP Faults.
   - Spielmechanik: 
     - Analysiere eine Fehlernachricht im XML-Format.
     - Lernziel: Strukturierte Fehlerbehandlung.
