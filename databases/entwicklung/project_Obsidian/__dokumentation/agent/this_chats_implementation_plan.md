Ziel des Projekts ist es eine Webseite auf basis von Obsidian-Ordner mit Inhalten aus der Obsidian-App.

Der User hat ThemeToggle, Suchleiste und er hat 2 Modi denn innerhalb der Obsidian Struktur git es ein getrenntes netz,
welches innerhalb der Obsidian Dateien nicht getrennt ist. Das heißt Alle Nodes die mit "__" beginnen sind teil eines
getrennten netzes wenn man es der Graph-View betrachtet. Wir nennen diesen Bereich Leitfaden denn sie sind durchnummeriert.
z.B. __01 *

Es gibt 3 Views verteilt auf 2 Sichtmodi des Users.
Sichtmode: Cores
Splitscreen: Graph-View | Node
Graph-View zeigt einen Graph-View mit welchem man zu den Nodes Steuern Kann.
Node Zeigt als was in Graph-View vom User ausgewählt wurde.
Bei jedem Neuladen der Webseite wird ein Counter zurück-Gesetzt. Dieser Counter ist dann dauerhaft die Session an das Node
im Graph-View gebunden, damit der User nachher nachvolziehen kann in welcher Reihenfolge er die Nodes betrachtet hat.

Sichtmode: Leitfaden
Splitscreen: Leitfaden | Node
Leitfaden ist ein ListView
