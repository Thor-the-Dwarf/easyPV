1. "IT-Grundschutz-Check"
   - Ziel: Fülle den Grundschutz-Baustein für einen Server aus.
   - Spielmechanik: 
     - Baustein: "SYS.1.1 Allgemeiner Server".
     - Anforderung: "Virenschutz installiert?" -> Ja/Nein.
     - Anforderung: "USB-Ports deaktiviert?" -> Ja/Nein.
     - Spieler muss die Maßnahmen als "Umgesetzt" markieren.
     - Messbarkeit: 100% Konformität mit BSI-Empfehlung.
   - Lernziel: Arbeiten mit dem IT-Grundschutz-Kompendium.

2. "Struktur-Analyse-Puzzler"
   - Ziel: Baue das Netz des Unternehmens nach (Strukturanalyse).
   - Spielmechanik: 
     - Objekte: Router, Server, Client, Gebäude.
     - Aufgabe: Zeichne die Abhängigkeiten.
     - Router verbindet Gebäude mit Internet.
     - Server hängt am Router.
     - Messbarkeit: Logisch korrektes Netzdiagramm.
   - Lernziel: Erster Schritt der BSI-Methodik.

3. "Schutzbedarf-Kalkulator"
   - Ziel: Bestimme den Schutzbedarf (Normal, Hoch, Sehr Hoch).
   - Spielmechanik: 
     - Asset: "Kantine-Speiseplan". -> Normal.
     - Asset: "Patientendaten". -> Sehr Hoch.
     - Asset: "Buchhaltung". -> Hoch.
     - Messbarkeit: Korrekte Einstufung nach CIA-Triade (Vertraulichkeit, Integrität, Verfügbarkeit).
   - Lernziel: Schutzbedarfsfeststellung.

4. "BSI-Lagebericht-Quiz"
   - Ziel: Erkenne die aktuellen Top-Bedrohungen.
   - Spielmechanik: 
     - Schlagzeilen laufen durch: "Ransomware", "DDoS", "Spam", "Klimawandel".
     - Spieler muss die Bedrohungen aus dem aktuellen BSI-Bericht "fangen".
     - Messbarkeit: Relevante Cyber-Bedrohungen erkannt.
   - Lernziel: Awareness für aktuelle Gefahrenlage.

5. "Zertifizierungs-Manager"
   - Ziel: Bringe das Unternehmen zur ISO 27001 Zertifizierung auf Basis von Grundschutz.
   - Spielmechanik: 
     - Schritte: 1. Strukturanalyse, 2. Schutzbedarf, 3. Modellierung, 4. Basis-Sicherheitscheck.
     - Spieler muss die richtige Reihenfolge einhalten.
     - Messbarkeit: Erfolgreicher Audit-Abschluss.
   - Lernziel: Ablauf einer BSI-Zertifizierung.
