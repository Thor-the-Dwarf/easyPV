3. "Passwort-Salting"
   - Ziel: Erschwere Rainbow-Table Attacken.
   - Spielmechanik: 
     - User A PW: "123456". Hash: "X".
     - User B PW: "123456". Hash: "X" (Schlecht!).
     - Aufgabe: Füge Salt hinzu.
     - User A: "123456" + "Salt1" -> Hash "Y".
     - User B: "123456" + "Salt2" -> Hash "Z".
     - Messbarkeit: Eindeutige Hashes trotz gleichem PW.
   - Lernziel: Schutz gespeicherter Passwörter.
