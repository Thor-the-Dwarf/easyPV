Anforderungen (UX-Game-Optimiert)
- Ziel: Transfer von logischen Abläufen in strukturierten Pseudocode zur Vorbereitung der eigentlichen Implementierung.
- Game-Mechanic: "The Logic Bridge". Du stehst vor einer Schlucht. Auf der einen Seite: Die menschliche Sprache ("Wenn der Kunde keine Pizza hat..."). Auf der anderen Seite: Der binäre Code. Du musst "Brückenelemente" aus Pseudocode-Bausteinen (`IF`, `WHILE`, `RETURN`) bauen, damit der Logik-Strom sicher rüberfließen kann.
- Challenge: 
  - Level 1 (Der einfache Weg): Baue eine `IF-ELSE` Abfrage für eine Altersprüfung.
  - Level 2 (The Loop-Gate): Konstruiere eine `FOR`-Schleife, um eine Liste von Bestellungen abzuarbeiten. Achte auf die Index-Grenzen.
  - Level 3 (The Complex Riddle): Ein Algorithmus für "Sitzplatz-Suche" (2D-Array). Baue die verschachtelten Schleifen so, dass kein Feld doppelt geprüft wird.

Arbeitspakete (UX-Game-Optimiert)
1. Interactive HUD und Scoring
- [ ] HUD-Elemente: "Logic-Integrity" (%), "Bridge-Stability", "Step-by-Step Execution Log".
- [ ] Scoring: "KISS-Bonus" (Keep It Simple, Stupid) für die kürzeste logische Lösung. Abzug bei "Logic-Leaks" (Endlosschleifen).

2. Screen-Layout (The Logic Canyon)
- [ ] `abyss-panorama -> natural-language-cloud -> construction-yard -> logic-flow-preview`.
- [ ] Natural-Language-Cloud: Hier schweben Textfetzen mit den Anforderungen.
- [ ] Construction-Yard: Ein Raster, in das Pseudocode-Blöcke wie Tetris-Steine eingeklickt werden.

3. Kernvisuals und Effekte
- [ ] Architektur-Sketch Ästhetik: Blaupausen-Hintergrund (Blueprint Blue), weiße Kreidestift-Linien, technische Zeichnungs-Symbole.
- [ ] "Blueprint-Lock": Wenn ein Block einrastet, gibt es einen haptischen "Snap" und eine weiße Konturlinie erscheint.
- [ ] "Logic-Flow": Ein kleiner Lichtpuls wandert nach dem Bauen testweise über die Brücke, um die Korrektheit zu prüfen.

4. Browser- und Device-Resilienz
- [ ] `background-image` mit einem Blueprint-Grid (shared_theme.css kompatibel).
- [ ] Mobile: "Long-Press to Rotate" (falls Blöcke gedreht werden müssen).

5. Content-Strategie (The Blueprints)
- [ ] 18 Übungen: Von "Maximum finden" bis "Bubble-Sort Logik".

6. QA-Paket
- [ ] Test: Sind die Pseudocode-Keywords (`WENN`, `SOLANGE`, `TUE`) sprachunabhängig logisch verständlich?
- [ ] Test: Wirkt der Übergang von Text zu Block-Design nahtlos?

7. Abnahme-Kriterien
- [ ] Der Spieler verankert logische Strukturen unabhängig von einer Programmiersprache.
- [ ] Das Interface wirkt intelligent, strukturiert und ästhetisch anspruchsvoll.
- [ ] Hoher Lernerfolg durch Visualisierung von Algorithmen.
