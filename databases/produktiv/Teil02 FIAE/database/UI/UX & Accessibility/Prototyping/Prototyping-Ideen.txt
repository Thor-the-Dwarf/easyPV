1. "Click-Path-Pilot"
   - Ziel: Verbinde Mockups zu einem interaktiven Fluss.
   - Spielmechanik: 
     - "Wenn ich auf 'Login' klicke, welches Bild soll als nächstes kommen?".
     - Lernziel: Interaktions-Logik.

2. "Interaction-Type Match"
   - Ziel: Wähle die richtigen Trigger.
   - Spielmechanik: 
     - "Click", "Hover", "Slash", "Double Click".
     - Lernziel: Vielfalt der User-Interaktionen.

3. "Prototype-Speedrun"
   - Ziel: Erstelle so schnell wie möglich einen lauffähigen Pfad.
   - Spielmechanik: 
     - Verbinde 5 Screens zu einem "Happy Path".
     - Lernziel: Rapid Prototyping.

4. "Transition-Master"
   - Ziel: Nutze Animationen sinnvoll.
   - Spielmechanik: 
     - "Slide-in" vs "Fade". Was passt besser zu einem Seitenwechsel?
     - Lernziel: Micro-Interactions und Motion Design.

5. "User-Journey-Validator"
   - Ziel: Finde Lücken im Klickpfad.
   - Spielmechanik: 
     - "Der Nutzer kann nicht zurück zum Menü". Füge einen Back-Button ein.
     - Lernziel: Vollständigkeit der User Flows.
